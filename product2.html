Hostlink

Hostlink Communication Protocol

Hostlink is a communication protocol designed for use with or between Omron PLCs (Programmable Logic Controllers). It is an ASCII-based protocol typically used for communication over RS-232 or RS-422. This protocol enables equipment in an industrial environment to communicate for programming or control purposes. The maximum allowed message size is 30 words per message. Larger messages can be sent through a process called "fragmentation," where the same slave device returns a series of messages to compile the entire response. Using the HostLink protocol, PLC host computers can transfer procedures, monitor PLC data areas, and control the PLC efficiently.
